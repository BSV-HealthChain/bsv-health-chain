# BSV HealthChain

BSV HealthChain is a peer-to-peer, FHIR-compliant decentralized healthcare management system built on the Bitcoin SV (BSV) blockchain.  
It enables secure health-data exchange, micro-payments, and decentralized medical compute using scalable on-chain transactions.

---

## Features

- **FHIR-standard**, interoperable health-data structure  
- **On-chain data anchoring** using BSV transactions  
- **Micro-payments** using satoshis  
- **ARC endpoints** for reliable transaction broadcasting  
- **Smart-contract logic** for healthcare workflows  
- **Off-chain pseudonymized compute** for medical AI processing  
- **Secure patient‚Äìdoctor interaction** and verifiable audit trails
